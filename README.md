# RegexRiot

Welcome to RegexRiot!

This is a utility regex-generating library designed to help simplify regex generation and parsing in Java.

This repo was created by [Rahul K](https://github.com/RK22000) and [Diego C](https://github.com/diego-ruben-cruz) as part of a class project for CS 152 at [San Jose State University](https://catalog.sjsu.edu/preview_course_nopop.php?catoid=10&coid=42155).

In more detail, RegexRiot is a library that allows for the creation of a `SimpleRiotString` object which can add different fields and parameters to aid in creating a string pattern which behaves in an append-only fashion similar to like libraries that exist for other languages.

RegexRiot was modeled on the exercises contained in [regextutorials.com](http://regextutorials.com/), and was inspired by the [magic-regexp](https://github.com/danielroe/magic-regexp) library that exists for JavaScript.
